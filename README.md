# BodyMini - 个人身体记录网页应用

一个极简、轻量、适合个人长期使用的单页应用（SPA），用于记录每天的饮食、作息与身体状态。

## 已实现功能（V1）

- 每日记录表单（包含睡眠、饮食、训练、补剂、身体感受等字段）
- 本地保存（`localStorage`）
- 历史记录列表
- 展开查看详情
- 编辑记录
- 删除记录
- 移动端与桌面端自适应布局

## 本地运行

这个项目没有后端和数据库，只需要静态文件服务即可。

### 方式 1：直接双击 `index.html`

大多数浏览器可直接运行。

### 方式 2：使用 Python 启动本地服务（推荐）

```bash
python3 -m http.server 8000
```

然后访问：

- <http://localhost:8000>

## 数据说明

- 数据存储在浏览器本地 `localStorage`
- 键名：`bodymini_daily_records_v1`
- 清空浏览器站点数据会导致记录丢失（可后续再加导出功能）

## 项目结构

- `index.html`：页面结构（标题、记录表单、历史列表）
- `style.css`：简洁响应式样式
- `app.js`：记录的增删改查与本地持久化逻辑
